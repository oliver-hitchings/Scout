# Prepare for an interview

1. Read the workspace profile, master CV, tracker entry, tailored material, and current application stage.
2. Re-check public source URLs and cite them with today's date.
3. Write `applications/<company-slug>/interview-prep.md` with a company refresher, stage-specific questions, evidence-backed talking points, honest gaps, and questions aligned with the user's priorities.
4. Mark the item interviewing where appropriate, add the named stage and a dated note, then commit the prep and tracker.

Never invent experience, metrics, contacts, or role facts. Use the user's locale and tone. Never send anything.
