# Record an application

1. Resolve the company, role, or stable ID in `data/opportunities.json`; ask if ambiguous.
2. Set the item to `applied` unless already interviewing, record today's applied date, ensure a completed Applied stage, and append the user's dated note.
3. Report which tailored CV, PDF, and outreach files exist. Never invent or send anything.
4. Commit only the tracker with `applied: <company>`.

Preserve all earlier notes, contacts, events, and stages. Use the workspace locale and tone.
