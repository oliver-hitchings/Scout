# Scout 0.1.0-beta.15

Beta 15 repairs the end-to-end tailored CV workflow while preserving existing private workspace files.

## Highlights

- Existing `cv.typ` sources remain visible when their PDF, quality record or tracker association is missing.
- The CV library shows useful source/PDF/quality state and provides a direct create action for tracked opportunities.
- Opportunity cards refresh as soon as a chat creates or updates CV files.
- Windows, macOS, Linux and the private VPS use a pinned, checksum-verified Typst 0.14.2 runtime supplied by Scout.
- Doctor reports the active Typst runtime source and version, and missing-runtime errors now recommend repairing Scout instead of installing an external command.

## Upgrade note

No private workspace migration is required. Existing CV sources and application folder names are retained unchanged. Refresh one page opened before the upgrade so the Beta 15 CV library code loads.
