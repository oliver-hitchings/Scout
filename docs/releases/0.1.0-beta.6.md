# Scout 0.1.0-beta.6

Scout beta.6 fixes the first-run search experience so a newly configured workspace can start finding opportunities immediately and stays configured after restart.

## Highlights

- Starts the first supervised job scan automatically after the required setup answers are saved.
- Adds a persistent **Scan now** button to the dashboard.
- Lets users enable, disable, and change the daily scan time from Scout settings.
- Persists setup completion in the private workspace so onboarding does not reopen after it is finished.
- Keeps optional AI/CV enrichment optional without preventing the first scan.

Scout never submits applications or sends outreach. All packages remain unsigned public betas; verify them against `checksums.txt` before installing.
