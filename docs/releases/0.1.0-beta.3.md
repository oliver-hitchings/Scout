# Scout 0.1.0-beta.3

Scout beta.3 improves source diagnostics, scheduled-scan visibility and phone-sized layouts while preserving the local-first workspace model.

## Highlights

- Distinguishes healthy, degraded and unavailable search sources without discarding valid partial results.
- Shows per-source scan health and the configured daily scan summary.
- Improves narrow-screen navigation, chat, setup and strong-match arrival layouts.
- Keeps all existing Scout character states, privacy boundaries and human review gates.

## Upgrade and privacy

This release does not change the workspace schema. Upgrading the application does not move, reset or publish the private workspace. The default workspace remains `%USERPROFILE%\Documents\Scout Workspace`.

The Windows installer is currently unsigned and may trigger Microsoft SmartScreen. Verify `checksums.txt` before running it. Scout never submits applications or sends outreach automatically.
