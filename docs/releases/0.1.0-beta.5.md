# Scout 0.1.0-beta.5

Scout beta.5 adds native-feeling packages for macOS and Linux while preserving the same local browser dashboard and private workspace model.

## Highlights

- Windows x64 installer, macOS Intel and Apple Silicon DMGs, Linux x64 DEB and portable tarball.
- Bundled Node runtimes and launchers that open the local Scout dashboard.
- Daily scans through Windows Task Scheduler, macOS launchd or Linux systemd user timers.
- Platform-specific installation, checksum, upgrade and uninstall guidance.

All packages remain unsigned public betas. Verify `checksums.txt`; Windows SmartScreen or macOS Gatekeeper may warn. Scout never submits applications or sends outreach.
