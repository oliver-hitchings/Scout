# Scout 0.1.0-beta.8

Scout beta.8 gives the Windows app a proper host process and notification-area lifecycle while completing provider discovery for the official Codex installation.

## Highlights

- Adds `Scout.exe` with Open, Restart, Check for updates, and Quit tray actions.
- Names the bundled server `ScoutRuntime.exe` instead of `node.exe` in Task Manager.
- Lets Quit keep daily scans enabled, disable them, or cancel.
- Offers an opt-in Start with Windows setting and prompts existing users only for this new setup section.
- Checks for updates shortly after launch and no more than daily, opening the verified GitHub release for manual installation.
- Detects authenticated Codex at `%LOCALAPPDATA%\Programs\OpenAI\Codex\bin\codex.exe` and uses that same executable for chat and scans.

Application updates continue to preserve the separate private Scout workspace.
