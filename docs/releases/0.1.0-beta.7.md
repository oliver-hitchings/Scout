# Scout 0.1.0-beta.7

Scout beta.7 fixes AI-provider detection for Windows desktop launches.

## Highlights

- Finds npm-installed Codex in `%APPDATA%\npm` even when Scout inherited an older or restricted desktop `PATH`.
- Adds standard Node.js locations so the Codex npm command shim can start reliably.
- Finds native Claude installations under `%USERPROFILE%\.local\bin`.
- Uses the same augmented environment for provider version and authentication checks.
- Clarifies that Scout requires the provider CLI login; a desktop-app login may be separate.

Scout never collects provider passwords or tokens. Authentication remains within the provider's official CLI flow.
