# Scout 0.1.0-beta.4

Scout beta.4 is the friend-ready Windows public beta. It makes first-run setup profession-neutral, requires genuinely approved profile/CV evidence before declaring setup complete, and puts the supervised first scan and daily schedule controls inside the app.

## Highlights

- Explains that searches come from the user's approved CV and preferences, not unrelated AI history.
- Guides Codex or Claude installation and authentication without collecting provider credentials.
- Runs the first supervised scan from Settings and blocks scheduling until a healthy scan has been reviewed.
- Uses persistent least-privilege Windows daily-task registration.
- Preserves existing beta workspaces without resetting tracker, calibration or application history.

## Installation and privacy

Follow [Install Scout on Windows](../INSTALL_WINDOWS.md). This release is unsigned and may trigger Microsoft SmartScreen; verify the published SHA-256 checksum before running it. Scout remains local-first, sends necessary task context only to the selected AI provider, and never submits applications or sends outreach.
